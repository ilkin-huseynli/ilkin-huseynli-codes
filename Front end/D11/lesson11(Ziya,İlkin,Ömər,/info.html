Proqramlaşdırma nə üçün lazımdır?

Proqramlaşdırma vasitəsiləri:

  • Veb saytlar və mobil tətbiqlər hazırlanır
  • Oyunlar yaradılır
  • Süni intellekt və robotlar inkişaf etdirilir
  • Bank sistemləri, e-dövlət xidmətləri qurulur
  • Məlumatlar analiz edilir və avtomatlaşdırma aparılır

Proqramlaşdırmanın əsas sahələri:

  • Veb Proqramlaşdırmanın
  • İnternet saytlarının və veb tətbiqlərin hazırlanmasıdır.
  • Frontend (ön hissə): HTML, CSS, JavaScript
  • Backend (arxa hissə): Python, Java, PHP, Node.js
  • Nümunə: Sosial şəbəkələr, onlayn mağazalar

Mobil Proqramlaşdırma:

  • Telefon və planşetlər üçün tətbiqlərin hazırlanması.
  • Android: Java, Kotlin
  • iOS: Swift
  • Çoxplatformalı: Flutter, React Native

Front End haqqda ətraflı məlumat üçün keçid edin.
