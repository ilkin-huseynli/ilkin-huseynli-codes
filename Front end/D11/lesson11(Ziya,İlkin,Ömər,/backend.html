Backend nədir?

Back-end nə iş görür?

  • Biznes məntiqi: Qaydalar, hesablamalar, icazələr
  • İstifadəçi autentifikasiyası: Login, qeydiyyat, tokenlər (JWT və s.)
  • API-lər: Front-end ilə əlaqə (REST, GraphQL)
  • Təhlükəsizlik: Şifrələmə, icazələr, hücumlardan qorunma
  • Performans və miqyaslama: Keş (cache), yük balansı

Populyar back-end dilləri və frameworklər:

  • JavaScript: Node.js (Express, NestJS)
  • Python: Django, Flask, FastAPI
  • Java: Spring Boot
  • C#: ASP.NET Core
  • PHP: Laravel
  • Go: Gin, Fiber

Back-end developer nə bilməlidir?

  • HTTP/HTTPS, REST, JSON
  • Məlumat bazası dizaynı (ERD)
  • Təhlükəsizlik əsasları (hashing, auth)
  • Git, Docker (üstünlükdür)
  • Testlər (unit/integration)
  • Deploy və server anlayışları (Linux, cloud)
Geri qayıtmaq üçün klikləyin.
